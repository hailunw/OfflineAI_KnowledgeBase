标题: 机能设计定義是什么？
内容: 
    相对于要件定义关注系统做什么， 
    基本设计关注有哪些模块，
    详细设计关注代码如何实现， 
    机能设计关注每个功能如何处理。
    本知识库文档，包含4个主要机能。
        1️⃣ 文档加载机能
        2️⃣ 知识库构建机能
        3️⃣ RAG检索机能
        4️⃣ 问答生成机能


标题: 文档加载机能是什么？
内容: 
    机能ID：F01
    机能名称：Markdown文档加载
    机能概要： 系统从指定目录读取 Markdown 文档， 并加载为 Document 对象。
    输入:
        项目	    内容
        目录路径	docs 目录
    处理流程:
        1 扫描 docs 目录
        2 查找所有 .md 文件
        3 使用 TextLoader 读取文件
        4 转换为 Document 对象
    输出:
    项目	            内容
    DocumentList	Markdown文档对象
    对应代码:
        DirectoryLoader
        TextLoader

标题: 知识库构建机能是什么？
内容: 
    机能ID：F02
    机能名称：向量数据库构建
    机能概要：系统将 Markdown 文档转换为向量，并构建 FAISS 向量数据库。
    输入:
        项目	        内容
        Document	Markdown内容
    处理流程
        1 MarkdownSplitter切分文档
        2 SentenceTransformer生成向量
        3 创建FAISS索引
        4 保存index.bin
        5 保存metadata.json

    流程图：    
        Markdown
           ↓
        split
           ↓
        embedding
           ↓
        FAISS
           ↓
        保存索引
    输出：
        项目	            内容
        index.bin	    FAISS索引
        metadata.json	chunk内容
    对应代码：
        init_rag_db()
        MarkdownSplitter
        SentenceTransformer
        faiss.IndexFlatIP
标题: RAG检索机能是什么？
内容: 
    机能ID：F03
    机能名称： RAG知识检索
    输入：
    项目	    内容
    用户问题	string
    处理流程：
        1 Query Rewrite
        2 生成向量
        3 FAISS搜索
        4 返回最相似chunk
    
    流程：    
        Query
         ↓
        Rewrite
         ↓
        Embedding
         ↓
        FAISS search
         ↓
        Result
    输出：
        项目	    内容
        知识内容	chunk
    对应代码：
        build_query_prompt()
        text_2_vector()
        rag_retrieval()

标题: 回答生成机能是什么？
内容: 
    机能ID： F04
    机能名称: 回答生成
    输入:
        项目	内容
        问题	Query
        知识	RAG结果
    处理流程:
        1 构建prompt
        2 调用LLM
        3 生成回答
    输出:
    项目	    内容
    Answer	文本回答
    对应代码:
        build_answer_prompt()
        llm_model.create_completion()
