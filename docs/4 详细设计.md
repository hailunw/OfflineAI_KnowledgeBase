
标题: 详细设计定義是什么？
内容: 对应的英语单词是 Detailed Design；目的是说明系统如何实现；包括类设计、函数设计、数据结构设计等。

标题: 离线知识库MarkdownSplitter模块
内容:
    作用： 按照 Markdown 标题切分文档
    类设计：class MarkdownSplitter
    方法：
        split()
        split_documents()
    
    流程：
        Markdown text
            ↓
        Regex 匹配标题
            ↓
        切分 chunk
            ↓
        生成 Document

标题: 离线知识库RAG Tool模块

    作用：初始化 SentenceTransformer
    模型：paraphrase-multilingual-MiniLM-L12-v2
    输出：embedding vector

标题: 详细设计阶段的成果物都有哪些？编写者是谁？
内容: 
    1. 详细设计书（詳細設計書）
        内容：各机能的处理流程、分支 / 异常逻辑、算法、调用关系、内部数据结构
        编写者：模块担当 SE、开发工程师
        依据：基本设计书、机能设计书、接口仕様
    2. 程序设计书（プログラム設計書）
       内容：类结构、方法定义、参数、处理顺序、返回值、内部逻辑
       编写者：开发工程师
       依据：详细设计书、接口设计书
    3. 数据库详细设计书（DB 詳細設計書）
       内容：表字段类型 / 长度 / 约束、索引、主键外键、SQL 设计、视图
       编写者：DBA、数据设计 SE、开发 SE
       依据：DB 基本设计书、详细机能逻辑
    4. 接口详细设计书（インタフェース詳細設計書）
       内容：报文格式、字段明细、错误码、通信方式、日志规则
       编写者：连携担当 SE、基盘 SE
       依据：接口基本设计、外部系统仕样
    5. 画面详细设计书（画面詳細設計書）
       内容：项目输入校验、事件处理、权限、按钮控制、错误提示
       编写者：前端 SE、画面担当 SE
       依据：画面基本设计、业务流程
    6. 共通部品详细设计书（共通部品詳細設計書）
       内容：通用类、方法仕样、调用规则、参数、返回格式
       编写者：基盘 SE、架构师
       依据：共通机能基本设计、开发规范
    7. 单体测试设计书（単体試験設計書）
       内容：测试用例、测试数据、期待结果、判定标准
       编写者：开发 SE（自测为主）或测试人员
       依据：详细设计书、程序设计书、业务规则
